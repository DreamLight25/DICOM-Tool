# 🦷 DICOM 信息提取与校正工具 (DICOM-Tool)

本工具专为医学临床科研设计，旨在解决临床影像资料整理中**姓名拼音难辨认**、**人工录入效率低**以及**大数据量处理慢**的痛点。

---

## 🌟 核心功能 (Features)

* **全自动提取**：秒级读取 DICOM 影像中的姓名、性别、检查日期等元数据。
* **年龄智能推算**：自动补齐影像中缺失的年龄标签。
* **序列去重**：一人一行，自动合并同次检查的成千上万张切片，无惧大数据量。
* **隐私安全**：提供本地运行方案，影像数据不经过网络，完全符合医疗合规要求。

## 📂 本地版运行指引 (Local Usage)

### 1. 下载工具

请在仓库右侧的 [Releases](https://www.google.com/search?q=https://github.com/DreamLight25/DICOM-Tool/releases) 页面下载最新的zip包并解压。

### 2. 环境配置 (仅需执行一次)

* **Windows 用户**：
1. 请确保已安装 Python 3.12 (推荐从 Microsoft Store 搜索安装)。
2. 进入 `DICOM tool (windows)` 文件夹，双击 `1.第一次使用请点我_安装环境 (Windows版).bat`。


* **macOS 用户**：
1. 进入 `DICOM tool (mac)` 文件夹。
2. 双击 `1. 第一次使用请点我_安装环境.command`。
3. *注：若提示权限问题，请在终端输入 `chmod +x ` 后将该文件拖入并回车。*



### 3. 开启工具

* **Windows 用户**：进入 `DICOM tool (windows)`，双击 `双击启动.bat`。
* **macOS 用户**：进入 `DICOM tool (mac)`，双击 `双击启动.command`。
